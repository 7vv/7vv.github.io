한글 인코딩 문제 해결 방안

발단

다른 서비스와 연동 할 일이 생겼는데 매번 연동 할때마다 서비스간 한글 인코딩 방식이 달라서 발생하는 문제점을 해결 해야만 했다.
같은 경험이 있는분에게 도움이 되거나 앞으로 종종 사용할것 같아서 여러 종류의 인코딩 문제를 해결하는 방법을 작성하기로 했다.

소개

  • 퍼센트어 한글로 변환하기.
  • 16진수 유니코드를 한글로 변환하기.
  • 이스케이프 문자열을 한글로 변환하기.
  • euc-kr 같은 인코딩 방식을 utf-8으로 디코딩 하기.

16진수 유니코드를 한글로 변환

javascript 내장함수를 이용해 한글로 변환할 수 있다. 16진수를 10진수로 변환한 뒤 해당 10진수 아스키코드를 문자열로 변환하면 된다.
아래는 소스코드에 사용한 내장 함수 설명이다. 보통 &#x으로 구분하지만 다른 구분자인 경우 ex) 0x 인 경우 &#x 부분을 바꿔주면 된다.

  • String.fromCharCode() 아스키코드를 문자로 변환하는 함수.
  • parseInt('16진수', 16) parseInt를 이용해 16진수를 10진수로 변환해줌.

사용 예제

  • 입력: &#xBE0C;&#xB79C;&#xB4DC;&#xD328;&#xC158;
  • 출력: 브랜드패션

퍼센트어 한글로 변환하기

대부분 알고 계시겠지만 javascript 내장함수인 encodeURI 으로 사용하면 됩니다.

  • 입력: %EA%B0%80%EB%82%98%EB%8B%A4%EB%9D%BC
  • 출력: 가나다라

이스케이프 문자열을 한글로 변환하기

이스케이프 문자열로 %uAC00 이나 \uAC00 같은 형태를 띄고있는데 unescape javascript 내장함수를 사용하면 됩니다

  • 입력1: \uAC00\uB098\uB2E4\uB77C\uB9C8\uBC14\uC0AC\uC544\uC790\uCC28\uCE74\uD0C0\uD30C\uD558\uD407\uD649\uB3D5
  • 입력2: %uAC00%uB098%uB2E4%uB77C%uB9C8%uBC14%uC0AC%uC544%uC790%uCC28%uCE74%uD0C0%uD30C%uD558%uD407%uD649%uB3D5
  • 출력1: 가나다라마바사아자차카타파하퐇홉돕
  • 출력2: 가나다라마바사아자차카타파하퐇홉돕

euc-kr 같은 인코딩 방식을 utf-8으로 디코딩 하기

문자 인코딩이 euc-kr 같은 경우 utf-8에서 외계어로 표시되는 경우가 있는데 이를 해결하려면 해당 인코딩 타입에 맞는 디코딩 과정을 거쳐야 합니다.
보통은 Node.js 에서 iconv 모듈을 사용하지만 설치할때 컴파일을 거치기때문에 모듈 설치 시간이 오래 걸리고 특수한 상황에서는 모듈 설치가 불가능합니다.
이때 iconv-lite 모듈을 사용하시면 되는데 iconv랑 사용법이 비슷합니다. 다만 인코딩, 디코딩 할 수 있는 타입이 줄어듭니다.

루프백 벨리데이션 다루기

루프백에선 기본적으로 벨리데이션 기능이 있습니다.
모델을 생성할 때 지정한 타입과 필수 여부가 벨리데이션 체크됩니다.

아래처럼 모델을 정의합니다.

testModel.json

이렇게 정의한 모델이 루프백 dataSource와 연결되어서 DataBase에 Create 할 때
벨리데이션이 동작하게 됩니다.

하지만 dataSource와 연결하지 않고 사용하는 리모트 메소드[REST API] 형태가 있습니다.
이 경우에는 직접 모델을 생성해서 벨리데이션 체크를 진행해야 합니다.

모델 안의 모델 벨리데이션 다루기

Input 모델이 아래와 같은 형태라면 모델안에 모델은 벨리데이션 체크가 이뤄지지 않습니다.
따라서 모델 안의 모델 벨리데이션을 해결하려면 한번 더 검사햐아합니다.

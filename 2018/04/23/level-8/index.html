#카탈란수

level-8-올바른 괄호

사용언어

  • python

문제 내용

  • N = 1일 경우에는 괄호가 () 1가지만 존재하므로 1을 리턴합니다.
  • N = 3일 경우에는 ((())), (())(), ()(()), ()()(), (()()) 5가지가 존재하므로 5를 리턴하면 됩니다.

올바른 괄호란 (())나 ()와 같이 올바르게 모두 닫힌 괄호를 의미합니다. )(나 ())() 와 같은 괄호는 올바르지 않은 괄호가 됩니다.
괄호를 이리저리 움직이며 올바른 괄호를 찾던 민호는 N개의 괄호쌍이 있을 때, 올바른 괄호를 만들 수 있는 경우의 수가 궁금해졌습니다.
괄호 쌍의 개수 N개가 주어졌을 때, 경우의 수를 반환하는 parenthesisCase 함수를 완성해 보세요.

문제 풀이

올바르게 닫힌 괄호를 의미하는데 이를 카탈란 수로 풀수있다.

해당 사진을 참고하면 이해가 쉽다.
이유는 x축 한칸 이동을 괄호 ( 추가하고 y축 한칸 이동을 ) 추가하는것으로 생각하면 된다.
이제 카탈란 수를 알고리즘으로 풀어 나가야하는데, 위키피디아에 자세하게 나와있다.

이런 공식인데 쉽게 풀자면 (2n)! / (n+1)!n! 이다. 여기서 n은 입력값이된다.

따라서 풀이 순서는

  • python 에는 팩토리얼 계산 함수가 있으므로 import해서 바로 사용하기.
  • 팩토리얼 계산 함수로 (2n)! / (n+1)!n! 공식 계산하기
  • 부동소수점 연산 문제로 나눗셈을 // 연산자 사용함

카탈란 수를 이용해서 해결하면 간단하다.

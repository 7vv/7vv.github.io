#Aws Summit Seoul 2018

Aws-Summit-2018 후기

Aws summit 행사때 부스를 준비하고 회사를 홍보하기 위해 참여했다.

입구

전날에는 사람이 더 많았는데 오늘은 한산한거 같다.

인공지능 세션

캔디 카메라로 찍었는데 화질이 좋지않다.

조금 일찍 도착했기에 9시~10시 사이에 인공지능 세션을 들었다.
윤석찬 태크 에반젤리스트 님이 발표하셨다.
인공지능 음성 비서 플랫폼인 Alexa Voice Service, Skill Kit 등을 설명해 주셨다. Aws Lamdba와 연동해서 음성 인식으로
EC2 리전을 바꾸거나 EC2를 멈추는 등 예제를 보여주셨는데 보다 편하게 음성인식을 사용할 수 있고 많은 서비스에 이용할 수 있겠다는 생각이 들었다.

AWS 알렉사 소개, 어떤 비즈니스 모델로 이용 가능한지 설명해주셨다.

원래는 듣고싶은 세션이 아니였으나 주제가 흥미로웠기에 들었다.
가장 재미있었던 세션이였던거 같다. 자꾸 기억에 남아 재미로 구현해보았다.

회사 부스

부스 관리와 홍보 목적으로 가서 모든 세션을 듣지못해 아쉬웠다.
그래도 같이오셨던 개발자 분들과 번갈아 가며 세션을 들을 수 있어서 좋았다.

후기

사진을 많이 찍지 못해 아쉬웠다. 휴대폰을 들어 카메라를 켜는게 왜이리 귀찮았던 걸까 궁금하다.
블로그 글을 처음 써보니 어떻게 써야할지 모르겠다.
노트에 정리했던 글을 시작으로 종종 써나갈 예정이다.
